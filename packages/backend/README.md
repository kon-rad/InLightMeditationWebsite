# prisma

to update database (for prototyping only, for production create migrations)

```npx prisma db push```

seed database

```npx prisma db seed```

run prisma studio

```npx prisma studio```

to start docker
```docker-compose up -d```

to init a prisma db
```npx prisma migrate dev --name "init" ```
